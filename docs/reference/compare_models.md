# Compare group counts and selection strategy across fitted models

Tabulates, for each fitted model (and SymPortal if present), the number
of groups, the selection strategy, and whether the model represents
within-sample mixing.

## Usage

``` r
compare_models(sp)
```

## Arguments

- sp:

  A `symbayes` object with one or more models fitted.

## Value

A data frame, printed and returned invisibly.

## Examples

``` r
if (FALSE)  compare_models(sp)  # \dontrun{}
```
